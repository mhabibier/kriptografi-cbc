# Cipher ROT-R ⊕ Key ⊕ Prev(IV) Plain
CARA KERJA ALGORITMA
[01] CBC XOR
Setiap byte plaintext di-XOR dengan ciphertext byte sebelumnya. Blok pertama menggunakan IV = 0xAA.
[02] SUBSTITUTE
Hasil XOR di-XOR lagi dengan karakter kunci. Kunci berulang (cyclic) jika lebih pendek dari plaintext.
[03] ROTATE LEFT
Byte digeser 1 bit ke kiri secara sirkular. Bit paling kiri masuk kembali dari kanan.
